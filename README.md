# AVE Landing Page

A landing page for AVE, a mobile prayer companion app that provides daily prayers, mysteries, novenas, and multilingual devotions.

## Built With

- HTML
- CSS
- Formspree (contact form)

## Features

- Responsive layout with a hero, features, testimonials, and contact sections
- Smooth scroll-triggered animations
- Working contact form via Formspree
- App Store and Google Play download buttons

## Live Site

Deployed via GitHub Pages.

## Setup

Clone the repo and open `index.html` in a browser. No build step required.
